## <img width="32" height="32" alt="image" src="https://github.com/user-attachments/assets/b957cf5a-fa2c-418a-be56-eefa17b70591" /> Windows XP UI Design Inspiration

This project is inspired by the **Windows XP (Luna theme)** interface, one of the most iconic operating system designs. The goal is to recreate its visual language and interaction patterns in a modern web environment using **HTML, CSS, and JavaScript**, starting from original design exploration in Figma.

---

## <img width="32" height="32" alt="image" src="https://github.com/user-attachments/assets/ea4a4f54-551b-4a4a-80f7-d00be2d77574" /> Design Goals

* Recreate the **classic Windows XP desktop experience**
* Study and reproduce **early 2000s UI patterns**
* Build a fully interactive **web-based desktop portfolio**
* Practice **UI design systems, layout structure, and component-based styling**
* Translate a **Figma prototype into functional frontend code**

## <img width="48" height="48" alt="image" src="https://github.com/user-attachments/assets/c99156a0-7ba9-40f6-a7a2-a4b43cb5f655" /> Core UI Components

The interface is built around the following components:

### Desktop

* Blue background (Bliss-inspired)
* Grid-aligned desktop icons
* Double-click interaction model

### Taskbar

* Fixed bottom navigation
* Start button with hover/active states
* Open applications indicator
* System tray and clock

### Windows

Each window contains:

* Title bar with controls (minimize, close)
* Content area
* Optional sidebar ("Common Tasks")
* Status bar

Rounded corners and gradient title bars are essential visual elements.

---

### Start Menu

Two-column layout:

Left column

* Applications
* Recent items

Right column

* System folders
* Control panel
* Log off / shutdown

## <img width="32" height="32" alt="image" src="https://github.com/user-attachments/assets/5a8b8637-d1f1-41b9-aeb4-3a61ac6afe49"/> Design Process

1. Research original Windows XP interface
2. Create UI components and layout in Figma
3. Define color system and spacing
4. Export assets and icons
5. Implement structure using semantic HTML
6. Style components using CSS
7. Add window interactions with JavaScript
